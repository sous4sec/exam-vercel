Atenção: não altere os nomes das colunas nem os nomes das abas. O sistema reconhece os dados pelo cabeçalho exato. Variações causarão erro na importação.
Regras de preenchimento
  • A planilha tem uma aba por turma — preencha cada aba com os alunos daquela turma.
  • Primeira linha de cada aba deve ser o cabeçalho com os nomes exatos das colunas.
  • Nenhuma célula das colunas obrigatórias pode ficar vazia.
  • A coluna ANO deve conter apenas o número: 1, 2 ou 3.
  • O arquivo deve estar no formato .xlsx.
